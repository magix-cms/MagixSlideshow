**MagixSlideshow** est un plugin de gestion de diaporamas (sliders) conçu spécifiquement pour **Magix CMS 4**. Il permet de créer, gérer et afficher des bannières dynamiques avec une intégration native au moteur de rendu du CMS.

## 🚀 Installation

1. Téléchargez et décompressez l'archive du plugin.
2. Placez le dossier `MagixSlideshow` dans le répertoire `plugins/` de votre installation.
3. Connectez-vous à l'administration de votre site.
4. Rendez-vous dans **Extensions** > **Gestionnaire**.
5. Cliquez sur le bouton d'installation automatique pour **MagixSlideshow**.

## 🛠 Utilisation & Layout

L'un des points forts de ce plugin est sa modularité :
* **Configuration :** Ajoutez vos images et textes via l'interface dédiée du plugin.
* **Placement flexible :** Une fois configuré, le module devient un **Widget** disponible dans l'outil **Layout**.
* **Gestion du positionnement :** Vous pouvez alors placer, déplacer ou ordonner votre diaporama sur votre page d'accueil (ou toute autre zone compatible) par simple glisser-déposer via le gestionnaire de mise en page.

## ✨ Fonctionnalités

* **Multi-langues :** Gérez des visuels et des textes différents pour chaque langue active.
* **SEO Friendly :** Optimisation automatique des balises `alt` et gestion des liens personnalisés.
* **Interface Magix native :** Utilisation des composants `MagixForms` et `MagixGallery` pour une expérience d'administration fluide.
* **Performance :** Chargement optimisé pour ne pas impacter le score Core Web Vitals.
